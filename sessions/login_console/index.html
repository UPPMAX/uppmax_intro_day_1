    Login via a terminal¶

    • Practice using the UPPMAX documentation
    • Understand what a console environment is

    • Understand what a prompt is

    Why?¶

    Using a terminal is powerful, where a remote desktop is clumsy. Copy-pasting text to a terminal on the remote desktop will quickly make you wonder if it cannot be done in a smarter way.

    A terminal and SSH clients¶

    A terminal is a text-only program that can do many things, for example, starting a program. An SSH client is a program that allows you to connect to another computer. Some SSH clients can run from a terminal or vice versa.

    Exercise 2: install an SSH client if needed¶

    Go to the UPPMAX documentation at https://docs.uppmax.uu.se, then answer these questions:

    Answer

    (optional) Exercise 4: use light graphics¶

    • Find the page about the program called eog
    I cannot find it. Where is it?

    You can find find it at https://docs.uppmax.uu.se/software/eog/

    • On the Pelle login node, create an example image or download an image
    How to do so?
    • Follow the tip in UPPMAX documentation about eog:
    convert -size 32x32 xc:transparent my.png
    • Use wget, e.g.
    wget https://raw.githubusercontent.com/UPPMAX/UPPMAX-documentation/refs/heads/main/docs/cluster_guides/img/pelle.jpg
    • (optional) On a Pelle login node, display the image using eog. Observe the error message
    How do I do so?

    In your terminal, type:

    eog [filename]

    where [filename] is the name of the file, for example:

    eog my.png

    The error message will be similar to this:

    (eog:2388308): Gtk-WARNING **: 11:15:47.357: cannot open display:
    • The UPPMAX page about eog mentions that one needs to login using SSH with X-forwarding. Go to the page that describes how to use SSH with X-forwarding
    I cannot find that page. Where is it?

    Searching for 'SSH with X-forwarding' results in finding https://docs.uppmax.uu.se/software/ssh_x_forwarding/

    • Log out of Pelle and login with X-forwarding enabled
    How to log out from Pelle?

    In the terminal, type:

    exit

    Closing the terminal also works.

    How to log in to Pelle using X-forwarding?

    Use the -X (a capital X) flag, e.g.

    ssh -X sven@pelle.uppmax.uu.se
    • On a Pelle login node, display an image using eog.
    How do I run it

    In your terminal, type:

    eog [filename]

    where [filename] is the name of the file, for example:

    eog my.png

    Press enter and you will the image pop up.
